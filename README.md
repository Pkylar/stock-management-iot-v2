# Sistem Manajemen Stok Berbasis IoT

## Struktur Project
```
stock-management-iot/
├── frontend/                 # Next.js Frontend
├── backend/                  # Laravel Backend
├── esp32-cam/               # Kode ESP32-CAM
├── database/                # SQL Files
└── README.md
```

## Teknologi
- Frontend: Next.js
- Backend: Laravel
- Database: MySQL
- IoT: ESP32-CAM (C++)

## Cara Menjalankan
1. Setup database MySQL
2. Jalankan backend Laravel
3. Jalankan frontend Next.js
4. Upload kode ke ESP32-CAM

## Fitur
- Login/Register
- Dashboard stok barang
- Tambah/Edit stok manual
- Riwayat stok masuk/keluar
- API untuk ESP32-CAM
